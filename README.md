# Netflix EDA Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The objective is to clean the dataset, analyze trends, visualize important patterns, and generate meaningful insights.

---

## Dataset

- Dataset: Netflix Movies and TV Shows
- Source: Kaggle
- File Used: `netflix_titles.csv`

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## Project Workflow

1. Data Loading
2. Data Inspection
3. Data Cleaning
4. Exploratory Data Analysis
5. Data Visualization
6. Insights Report

---

## Key Insights

- Movies are more common than TV Shows on Netflix.
- The United States has the largest number of Netflix titles.
- Netflix content increased rapidly after 2015.
- Drama and International Movies are among the most popular genres.
- TV-MA and TV-14 are the most common content ratings.
